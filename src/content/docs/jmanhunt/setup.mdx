Willkommen beim Setup. Folge diesen Schritten, um das Projekt zu starten.

<Aside type="caution">
  Stelle sicher, dass du mindestens **Node.js** v18 installiert hast, bevor du beginnst.
</Aside>

## Installationsschritte

<Steps>

1.  **Repository klonen**
    Lade dir den Quellcode herunter:
    ```bash
    git clone [https://github.com/dein-nutzer/jmanhunt.git](https://github.com/dein-nutzer/jmanhunt.git)
    ```

2.  **Abhängigkeiten installieren**
    Gehe in den Ordner und installiere die Pakete:
    ```bash
    npm install
    ```

3.  **Entwicklungsserver starten**
    Starte das Projekt lokal:
    ```bash
    npm run dev
    ```

</Steps>
